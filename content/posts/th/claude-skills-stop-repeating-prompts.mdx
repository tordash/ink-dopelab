## ปัญหาที่ทุกคนมี แต่ไม่รู้ว่ามี

ทุกวัน คุณเปิด AI แล้วพิมพ์อะไรบ้าง?

"สรุปให้หน่อย"
"เขียนแคปชันให้"
"ช่วยแปลเป็นไทย"
"วิเคราะห์ข้อมูลนี้"

แล้วทุกครั้ง คุณต้องอธิบายอีกว่า:
- format แบบไหน
- tone ยังไง
- ภาษาไทยหรืออังกฤษ
- ใส่ hashtag มั้ย

prompt เดิม ทุกวัน ทุกสัปดาห์ ซ้ำแล้วซ้ำอีก

ถ้าคุณจ้างพนักงานมาทำงานเดิมทุกวัน แต่ต้องสอนใหม่ทุกเช้า คุณจะทนได้นานแค่ไหน?

## Claude Skills คือ "การฝึก" ที่ AI จำได้ตลอด

Skills คือไฟล์คำสั่งที่คุณเขียนครั้งเดียว แล้ว Claude จำได้ตลอดไป

เรียกใช้ด้วย `/ชื่อ-skill` — จบ

ไม่ต้องพิมพ์ prompt ยาวๆ ไม่ต้องอธิบายซ้ำ ไม่ต้องกลัวลืม format

## ตัวอย่าง Skills ที่เราใช้ทุกวัน (ของจริง)

### /news-card — สร้างโพสต์ข่าว
พิมพ์แค่: `/news-card "Anthropic จะ IPO"`

Claude ทำให้ทั้งหมด:
- หาข้อมูลข่าว
- เขียน headline + subtitle
- gen รูป NB2
- render เป็น PNG 1080x1350
- พร้อมโพสต์

เมื่อก่อนทำเอง: 2 ชม.
ตอนนี้: 5 นาที

### /caption — เขียนแคปชัน
ไม่ต้องบอก format ทุกครั้ง Skill จำได้ว่า:
- ใช้ PAS framework
- Hook → ข่าว → วิเคราะห์ → CTA
- Hashtag แบบไหน
- tone เสียงต่อ ไม่ใช่ corporate

### /deploy — อัปโหลดงานขึ้น server
พิมพ์แค่ `/deploy` แล้วบอกว่าไฟล์ไหน
Claude rsync ขึ้น server, เช็ค SSL, verify ว่าเปิดได้จริง

ไม่ต้องจำ IP, path, SSH key ทุกครั้ง

### /rrr — สรุป session ก่อนจบงาน
Claude สรุปว่าทำอะไรไปบ้าง สกัด lesson learned บันทึกลง brain

ตอนนี้เรามี 30+ Skills ทำงานอยู่

## วิธีสร้าง Skill แรกของคุณ

ง่ายกว่าที่คิด:

**ขั้น 1:** คิดว่าอะไรที่คุณพิมพ์ให้ AI ซ้ำทุกวัน

**ขั้น 2:** เขียนเป็นไฟล์ SKILL.md

```
# /สรุป-ข่าว

สรุปข่าว AI 24 ชม. ล่าสุด
- ค้นหาจาก 5 แหล่ง
- เลือก 3 ข่าวที่เกี่ยวกับ SME ไทย
- เขียนสรุปทีละข่าว 3 บรรทัด
- ภาษาไทย ไม่ใช้ศัพท์ยาก
```

**ขั้น 3:** วางไว้ใน `.claude/skills/สรุป-ข่าว/SKILL.md`

**ขั้น 4:** พิมพ์ `/สรุป-ข่าว` ได้เลย

ใช้เวลาตั้งค่า 5 นาที ประหยัดเวลาทุกวันตลอดไป

## ทำไมเรื่องนี้สำคัญกว่าที่คิด

Skill ไม่ใช่แค่ shortcut

มันคือ **ระบบ** ที่ทำให้ AI ทำงานเหมือนพนักงานจริงๆ

พนักงานที่:
- จำทุกอย่างที่คุณสอน
- ไม่ลืม ไม่ลาออก ไม่ต้องสอนซ้ำ
- ทำงานได้ 24 ชม.
- ต้นทุน 20 เหรียญต่อเดือน

คนที่สร้าง Skills เยอะ = คนที่ได้ประโยชน์จาก AI มากที่สุด

เริ่มจาก 1 Skill ก่อน แล้วจะติดใจ
