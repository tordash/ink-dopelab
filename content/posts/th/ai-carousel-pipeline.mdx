## โพสนี้ทำด้วย AI ทั้งหมด

Caption ที่คุณอ่านอยู่ → AI เขียน
รูปที่คุณเห็น → AI สร้าง
การโพสขึ้น social → AI ทำ

**ไม่ได้ใช้ Canva ไม่ได้จ้างกราฟิก**

ใช้แค่ 3 เครื่องมือ ต่อกันเป็น pipeline:

```
✍️ Claude Code  →  🎨 Gemini (Nano Banana)  →  📤 Blotato
  เขียน caption      สร้างรูป                    โพส FB+IG
```

## Step 1: Claude Code เขียน Caption

บอก AI ว่าอยากโพสเรื่องอะไร:

```
"เขียน carousel caption 5 slides เรื่อง AI บริหารสต็อควัตถุดิบร้านอาหาร
 — โทนเล่าเรื่อง, case study จริง, มี data จริง
 — CTA: Swipe ดูวิธีทำ"
```

AI เขียนมาทั้ง:
- Hook (slide 1) ที่ดึงความสนใจ
- เนื้อหา (slide 2-4) ที่มี data + insights
- CTA (slide 5) ที่กระตุ้น action
- Hashtags ที่เหมาะสม

ผมอ่าน แก้ 2-3 จุด → approve

## Step 2: Gemini สร้างรูป

นี่คือส่วนที่มหัศจรรย์ที่สุด

**Gemini (Nano Banana 2)** สร้างรูปได้แบบ photorealistic — ไม่ใช่ AI look แบบเดิมที่ดูปลอมๆ

### ข้อดีของ Gemini สำหรับ carousel:

**1. เขียนภาษาไทยบนรูปได้ชัดมาก**
นี่คือ game changer AI ส่วนใหญ่เขียนภาษาไทยบนรูปแล้วเบลอ/ผิด แต่ Gemini ทำได้ดีมาก → ไม่ต้องเปิด Canva มาใส่ text ทีหลัง

**2. Photorealistic**
รูปที่ออกมาดูเหมือนถ่ายจริง ไม่ใช่ "AI art" ที่คนเลื่อนผ่าน

**3. ควบคุม style ได้**
```
"ภาพ cinematic cafe, คนถือ iPhone มี terminal code สีเขียว
 มีข้อความภาษาไทย 'สั่งงาน AI จากมือถือ' ตรงกลาง
 โทนสีอุ่น film grain เล็กน้อย"
```

<Callout variant="info" title="Cost">
Gemini API คิดค่า image generation ต่อรูป ราคาประมาณ ฿3-5 ต่อรูป ถูกมากเทียบกับจ้างกราฟิก
</Callout>

## Step 3: Blotato โพสอัตโนมัติ

Blotato เป็นเครื่องมือที่โพส content ไปหลาย platform พร้อมกัน:

- ✅ Facebook Page
- ✅ Instagram
- ✅ TikTok (video)
- ✅ YouTube (shorts)

Upload รูป carousel + caption → กด post → ขึ้นทุก platform

### ข้อดี
- **Schedule ได้** — ตั้งเวลาโพสล่วงหน้า
- **Analytics** — ดูผลลัพธ์รวมจากทุก platform
- **AI voice** — ฟรี ไม่เสีย credit

## Timeline จริง

| ขั้นตอน | เวลา |
|---------|------|
| Brief AI + เขียน caption | 10 นาที |
| Review + แก้ caption | 5 นาที |
| AI สร้างรูป 5 slides | 10 นาที |
| Review + แก้รูป (ถ้าต้อง) | 5 นาที |
| Upload + โพส via Blotato | 5 นาที |
| **รวม** | **~30 นาที** |

เมื่อก่อนใช้ 2-3 ชั่วโมง ต่อ 1 carousel (หา stock photo, เปิด Canva, จัด layout, ใส่ text...)

## เปรียบเทียบค่าใช้จ่าย

| วิธี | ค่าใช้จ่ายต่อ carousel | เวลา |
|------|---------------------|------|
| จ้างกราฟิก | ฿500-1,500 | 1-3 วัน |
| ทำเอง (Canva) | ฿0 (แต่ใช้เวลา) | 2-3 ชม. |
| **AI Pipeline** | **~฿20** (API cost) | **30 นาที** |

## สิ่งที่เรียนรู้

### AI ไม่ได้แทนที่ creativity
AI ทำงานซ้ำๆ ได้เร็วมาก แต่ **ไอเดีย** ยังมาจากคน ผมยังเป็นคน decide ว่าจะโพสเรื่องอะไร, โทนแบบไหน, target ใคร

### Quality ยังต้อง review
AI สร้างเร็ว แต่ไม่ perfect ทุกครั้ง รูปบางรูปต้อง re-generate, caption บางจุดต้องแก้ — แต่ถึงรวมเวลา review เข้าไป ก็ยังเร็วกว่าทำมือหลายเท่า

### Pipeline สำคัญกว่า tools
เครื่องมือเปลี่ยนได้ (วันนี้ Gemini, พรุ่งนี้อาจเป็นตัวอื่น) แต่ **flow การทำงาน** ต่างหากที่สำคัญ: brief → write → design → publish

<Callout variant="success" title="Bottom Line">
ถ้าคุณทำ social content อยู่แล้ว ลองเปลี่ยนบางขั้นตอนมาใช้ AI ไม่ต้องเปลี่ยนทั้งหมดทีเดียว เริ่มจากให้ AI ช่วยเขียน draft caption → คุณ review → ค่อยเพิ่ม AI สร้างรูปทีหลัง
</Callout>
