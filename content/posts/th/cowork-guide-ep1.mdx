## "ก็แค่ AI chatbot อีกตัวนึง?"

![Cowork with AI — ทำงานคู่กับ AI แบบเพื่อนร่วมงาน](/images/blog/cowork-ep1.jpg)

ถ้าคุณเคยใช้ ChatGPT หรือ Claude.ai แบบพิมพ์ถาม-ได้ตอบ แล้วรู้สึกว่า...

- ต้อง copy-paste ข้อมูลเข้าไปทุกครั้ง
- ถามเสร็จก็ต้อง copy ผลลัพธ์กลับมาใส่ไฟล์เอง
- สั่งให้ "จัดไฟล์" หรือ "สร้าง Excel" ไม่ได้จริง
- พอเปิดแชทใหม่ มันก็ลืมทุกอย่าง

**คุณไม่ได้ใช้ผิด** — เครื่องมือพวกนั้นมันถูกออกแบบมาแค่ "ตอบคำถาม" จริงๆ

แต่ถ้ามี AI ที่ **เปิดไฟล์ในเครื่องคุณได้** แก้ไขเอกสารให้ได้ สร้างไฟล์ใหม่ได้ เปิดเว็บค้นข้อมูลได้ — ทั้งหมดนี้แค่พิมพ์บอกเป็นภาษาคน?

นั่นคือ **Claude Cowork**

---

## Cowork คืออะไร? (3 บรรทัดจบ)

1. **AI agent** ที่ทำงาน **ในเครื่องคุณ** — ไม่ใช่แค่ตอบในแชท
2. คุณ **ชี้โฟลเดอร์** ให้มัน → มันอ่าน แก้ สร้างไฟล์ให้ได้เลย
3. บอกงานที่อยากได้ → **ปล่อยให้มันทำ** → กลับมาดูผลลัพธ์

ถ้าเปรียบเทียบ: ChatGPT เหมือน **ถามเพื่อน** ที่รู้เยอะ แต่ Cowork เหมือน **จ้างเลขา** ที่ลงมือทำงานให้จริงๆ

---

## Cowork vs Chat vs Claude Code

| | Claude Chat | Claude Cowork | Claude Code |
|---|---|---|---|
| **ใครใช้** | ทุกคน | เจ้าของธุรกิจ, Marketer, Creative | Developer |
| **หน้าตา** | แชทในเว็บ | แอป Desktop (GUI สวย) | Terminal (จอดำ) |
| **เข้าถึงไฟล์** | ต้อง upload ทีละไฟล์ | เปิดโฟลเดอร์ได้เลย | เข้าถึงทั้ง project |
| **สร้างไฟล์ได้** | ไม่ได้ (แค่แสดงผล) | ได้ — Excel, PowerPoint, เอกสาร | ได้ — code ทุกภาษา |
| **เปิดเว็บได้** | ไม่ได้ | ได้ (Claude in Chrome) | ต้อง config เอง |
| **ตั้งเวลาอัตโนมัติ** | ไม่ได้ | ได้ (`/schedule`) | ไม่ได้ |
| **ต้องเขียน code** | ไม่ต้อง | **ไม่ต้องเลย** | ต้อง |

<Callout variant="info" title="สรุปง่ายๆ">
Chat = ถามตอบ | Cowork = ทำงานแทน | Code = เขียนโปรแกรม

ถ้าคุณไม่ใช่ developer แต่อยาก AI ทำงานจริงๆ → **Cowork คือคำตอบ**
</Callout>

## เริ่มต้นใน 5 ขั้นตอน

### ขั้นที่ 1 — โหลดแอป

ไปที่ [claude.com/download](https://claude.com/download) → เลือก Mac หรือ Windows → ติดตั้งเหมือนแอปทั่วไป

### ขั้นที่ 2 — Login

เปิดแอป → Sign in ด้วย account Claude ที่มีแพลน Pro ขึ้นไป

### ขั้นที่ 3 — สลับไป Cowork Mode

ในแอปจะเห็นตัวเลือก **"Chat"** กับ **"Cowork"** — กดเลือก **Cowork**

### ขั้นที่ 4 — สร้าง Task แรก

กดไปที่แท็บ **Tasks** → พิมพ์สิ่งที่อยากให้ทำ เช่น:

> "จัดไฟล์ในโฟลเดอร์ Downloads ให้เป็นระเบียบ แยกตามประเภทไฟล์"

### ขั้นที่ 5 — ให้สิทธิ์เข้าโฟลเดอร์

Claude จะถามว่าอยากให้เข้าถึงโฟลเดอร์ไหน → เลือกโฟลเดอร์ที่ต้องการ → **กด Allow**

แค่นี้ Claude จะเริ่มทำงานให้เลย

## ลองเลย: Task แรก — "จัดไฟล์ใน Downloads"

ลองทำตามจริงๆ ดูครับ ใช้เวลาไม่ถึง 5 นาที

### สิ่งที่จะเกิดขึ้น

คุณมีโฟลเดอร์ Downloads ที่ไฟล์กระจัดกระจาย — PDF ปนกับรูป ปนกับ Excel ปนกับ zip

พิมพ์บอก Cowork ว่า:

> "ดูไฟล์ทั้งหมดในโฟลเดอร์นี้ แยกเป็นโฟลเดอร์ย่อยตามประเภท — เอกสาร, รูปภาพ, สเปรดชีต, อื่นๆ แล้วย้ายไฟล์เข้าไปให้"

### Before
```
Downloads/
├── รายงานQ4.pdf
├── สัญญาเช่า.pdf
├── IMG_2847.jpg
├── screenshot.png
├── ยอดขาย-มกราคม.xlsx
├── presentation.pptx
├── invoice-feb.pdf
├── logo-v3.png
├── backup.zip
└── ... อีก 200 ไฟล์
```

### After (Claude จัดให้)
```
Downloads/
├── 📄 เอกสาร/
│   ├── รายงานQ4.pdf
│   ├── สัญญาเช่า.pdf
│   └── invoice-feb.pdf
├── 🖼️ รูปภาพ/
│   ├── IMG_2847.jpg
│   ├── screenshot.png
│   └── logo-v3.png
├── 📊 สเปรดชีต/
│   └── ยอดขาย-มกราคม.xlsx
├── 📽️ พรีเซนเทชัน/
│   └── presentation.pptx
└── 📦 อื่นๆ/
    └── backup.zip
```

**ใช้เวลา:** ไม่ถึง 1 นาที — ไม่ว่าจะมี 50 หรือ 500 ไฟล์

<Callout variant="success" title="ลองทำตาม">
เปิด Cowork → ชี้ไปที่โฟลเดอร์ Downloads ของคุณ → สั่งจัดไฟล์ → ดูผลลัพธ์

ถ้าไม่อยากให้ย้ายจริง ลองสั่งแค่ "ดูไฟล์ทั้งหมดแล้วเสนอวิธีจัดกลุ่ม" ก่อนก็ได้ — Claude จะเสนอแผนให้ review ก่อนลงมือ
</Callout>

## Cowork ทำอะไรได้อีก?

นอกจากจัดไฟล์ ตัวอย่างงานที่ Cowork ทำได้จริง:

### สร้างเอกสาร
- สรุป meeting notes จากไฟล์ที่บันทึกไว้ → สร้างเป็น PowerPoint
- อ่าน data จาก CSV → สร้าง Excel report พร้อมกราฟ
- รวมข้อมูลจากหลายไฟล์ → สร้างรายงานสรุปไฟล์เดียว

### ค้นหาและวิเคราะห์
- อ่านไฟล์ PDF ยาวๆ แล้วสรุปให้
- เปรียบเทียบข้อมูลระหว่าง 2-3 ไฟล์
- หาตัวเลขที่ต้องการจากเอกสารหลายร้อยหน้า

### เปิดเว็บ (Claude in Chrome)
- ค้นข้อมูลคู่แข่ง แล้วสรุปเป็นตาราง
- ดึงข้อมูลจากเว็บมาเก็บเป็นไฟล์
- กรอกฟอร์ม, อ่านอีเมล, เช็คข้อมูล

### ทำงานอัตโนมัติ (/schedule)
- ทุกเช้า: สรุปอีเมลสำคัญ → เก็บเป็นไฟล์
- ทุกสัปดาห์: รวมข้อมูลยอดขาย → สร้าง report
- ทุกวันจันทร์: เช็คปฏิทิน → สรุปนัดหมายของสัปดาห์

## ข้อควรรู้ก่อนใช้

| ข้อจำกัด | รายละเอียด |
|---|---|
| **ต้องเปิดแอปค้าง** | ปิดแอป = Claude หยุดทำงาน |
| **ไม่ sync ข้ามเครื่อง** | ประวัติการทำงานเก็บไว้เฉพาะในเครื่องนั้น |
| **เริ่มใหม่ทุก session** | Claude ไม่จำ session ก่อนหน้า (แต่อ่านไฟล์ที่มีได้) |
| **ลบไฟล์ต้อง approve** | Claude จะถามก่อนทุกครั้งที่จะลบอะไร |
| **ไม่ใช่ของฟรี** | ต้องมีแพลน Pro ($20/เดือน) ขึ้นไป |

## สรุป

Claude Cowork ไม่ใช่ chatbot ที่ตอบคำถาม — มันเป็น **AI agent** ที่ลงมือทำงานจริงในเครื่องคุณ

**3 สิ่งที่ต้องจำ:**
1. **ไม่ต้องเขียน code** — พิมพ์ภาษาคนได้เลย
2. **ทำงานกับไฟล์จริง** — อ่าน แก้ สร้าง ย้าย ได้หมด
3. **ตั้งเวลาอัตโนมัติได้** — ไม่ต้องสั่งซ้ำทุกวัน

ถ้าคุณเป็นเจ้าของธุรกิจ เป็น Marketer เป็น Creative ที่เคยรู้สึกว่า "AI มันก็แค่ตอบๆ ไป" — ลอง Cowork ดูครับ มันจะเปลี่ยนวิธีคิดของคุณเกี่ยวกับ AI ไปเลย
