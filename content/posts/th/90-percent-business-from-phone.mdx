## ผู้ชายคนนี้บริหารธุรกิจ 90% จากมือถือ

ไม่ได้นั่งหน้าคอม ไม่ต้องอยู่ออฟฟิศ แค่ใช้ Claude Code + ฟีเจอร์ใหม่ที่ชื่อ **Remote Control**

เขาชื่อ **Nick Puru** — เคยทำรายได้ให้ลูกค้ารวม **$5 million+** ตอนนี้สอนคนสร้าง AI Agency มี community **18,000+ คน**

![บริหารธุรกิจจากมือถือ — AI interface ลอยเหนือหน้าจอ](/images/blog/remote-control-hero.jpg)

ผมดูคลิปเต็ม 16 นาทีแล้ว สรุปทุก use case ให้ครับ

## Remote Control คืออะไร?

ปกติ Claude Code ต้องเปิดบนคอม แต่ Remote Control ทำให้คุณ:

1. **สแกน QR code** จากมือถือ
2. **เชื่อมต่อ** กับ session ที่เปิดอยู่บนคอม
3. **สั่งงาน AI** จากที่ไหนก็ได้

ให้นึกภาพแบบนี้ — เหมือนมี **รีโมท** ชี้ไปที่ **TV** (คอม) แต่กดจากโซฟา (มือถือ)

![Remote Control Concept — มือถือ สแกน QR เชื่อมคอม สั่งงาน AI Operating System](/diagrams/remote-control-concept.svg)

**ข้อจำกัดที่ต้องรู้:** คอมต้องเปิดอยู่ ยังไม่ใช่ cloud — ต้องมีเครื่อง host session ไว้

## Use Case 1: Marketing Reports ทันที

**ปัญหา:** เจ้าของธุรกิจเสีย **3-4 ชั่วโมง/สัปดาห์** แค่ดึงรายงานจาก ad accounts + CRM

![Before vs After — จากนั่งดึงข้อมูลเอง สู่การสั่งจากมือถือ](/images/blog/remote-control-before-after.jpg)

**หลังใช้ Remote Control:**

- พิมพ์จากมือถือ → Claude ดึง metrics จากทุก platform
- สรุปเป็น **daily brief** อัตโนมัติ
- ถ้า cost-per-lead พุ่ง → **แจ้งเตือนทันที**
- สั่ง pause campaign + shift budget ได้เลยจากมือถือ

**ผลลัพธ์:** จาก 4 ชม./สัปดาห์ → **ไม่กี่วินาที**

## Use Case 2: ตอบ Lead ทันที

**ปัญหา:** Lead เข้ามา แต่ใช้เวลา 3-4 ชั่วโมงกว่าจะตอบ ลูกค้าของ Nick เคย **เสีย 30% ของ leads** เพราะตอบช้า — คิดเป็นเงินหลายแสน

![Lead Response Speed — ตอบใน 5 นาที ปิดดีลสูงกว่า 21 เท่า](/images/blog/remote-control-lead-speed.jpg)

**หลังใช้ Remote Control:**

- Claude **qualify lead** อัตโนมัติ
- Lead ร้อน → ส่งต่อทีมขายพร้อม context ครบ
- Log เข้า CRM
- แจ้ง notification มือถือ
- **นัด call อัตโนมัติ**

**ผลลัพธ์:** จาก 3-4 ชั่วโมง → **ทันที**

ข้อนี้สำคัญมาก — research บอกว่า lead ที่ได้รับการตอบภายใน 5 นาที มีโอกาสปิดสูงกว่า 21 เท่า เทียบกับ 30 นาที

## Use Case 3: Meeting Intelligence

ทุกการประชุมถูก transcribe อัตโนมัติผ่าน **Fireflies MCP** แล้วถาม Claude ได้เลย:

- "ลูกค้า object อะไรบ่อยสุด?"
- "ทีม sales กับ marketing เห็นต่างตรงไหน?"
- "สรุป action items จาก meeting วันนี้"

![Meeting Intelligence — AI เชื่อมจุดข้าม meetings ข้ามทีม](/images/blog/remote-control-meeting-intelligence.jpg)

ไม่ใช่แค่ transcribe — แต่ **เชื่อมจุด** ข้าม meetings ข้ามทีม เหมือนมี analyst ที่ฟังทุกประชุมแล้วบอกว่า "เรื่องนี้ทีม A พูดตรงกับ ทีม B นะ"

## ลองเริ่มใช้ Remote Control

ถ้าใช้ Claude Code อยู่แล้ว ลองเริ่มจาก:

1. **เปิด Claude Code บนคอม** → จะเห็น QR code สำหรับ Remote Control
2. **สแกนจากมือถือ** → เชื่อมต่อ session
3. **เริ่มจากคำสั่งง่ายๆ** — เช่น "สรุปไฟล์นี้ให้หน่อย" หรือ "ดึงข้อมูลจาก X"
4. **ค่อยขยาย** → ต่อ MCP servers (Notion, CRM, Fireflies) เพื่อให้ทำงานจริงได้

![เริ่มใช้ Remote Control — 4 ขั้นตอนจากง่ายไปขั้นสูง](/diagrams/remote-control-getting-started.svg)

แต่จำไว้: **สร้าง AI Operating System ก่อน** → แล้วค่อย remote มัน รีโมทไร้ค่าถ้าไม่มี TV
