## เมื่อก่อนผมพูดว่า "รอ 2-3 สัปดาห์"

![AI Website Builder — สร้างเว็บไซต์ใน 1 ชั่วโมง](/images/blog/website-1-hour.jpg)

ลูกค้ารายหนึ่งถามผมว่า "อยากได้หน้าเว็บสำหรับรับสมัครงานพาร์ทไทม์ ทำได้ไหม?"

คำตอบแบบเดิมของผมคือ รอ designer sketch, รอ approve, รอ developer build, รอ test, รอ deploy — **2-3 สัปดาห์** ถ้าโชคดี

วันนั้นผมลองทำวิธีใหม่แทน

## เครื่องมือที่ใช้จริง

ผมใช้ **Claude Code** เป็นหลัก — เปิด terminal แล้วคุยกับ AI ตรงๆ ได้เลย ไม่ต้องเขียน code เองสักบรรทัด

**ทางเลือกอื่นถ้ายังไม่คุ้น terminal:**
- **v0.dev** (Vercel) — พิมพ์ prompt ออกมาเป็น React component ได้เลย ใช้ฟรีในวงจำกัด
- **Bolt.new** — สร้างเว็บได้เร็ว เหมาะกับโปรเจ็คเล็กๆ ที่ต้องการเร็ว
- **Cursor** — IDE ที่มี AI built-in ดีสำหรับคนที่เริ่มเขียน code ได้บ้าง

## บอก AI เหมือนคุยกับคนจริงๆ

ผมไม่ได้เขียน code เป็นสักบรรทัด แค่ส่ง brief ให้ Claude Code แบบนี้:

```
สร้าง landing page สำหรับรับสมัครพนักงาน part-time ภาษาไทย

ธุรกิจ: ร้านอาหารไทย ย่านสุขุมวิท เปิด 10.00-22.00 น.
ตำแหน่ง: พนักงานเสิร์ฟ, ล้างจาน, แคชเชียร์
กลุ่มเป้าหมาย: นักศึกษา แม่บ้าน คนหางานพาร์ทไทม์

สิ่งที่ต้องมี:
- Header พร้อม logo placeholder และชื่อร้าน
- Section บอกว่ารับสมัครตำแหน่งอะไร เงินเดือนเท่าไหร่
- ฟอร์มสมัครงาน: ชื่อ, เบอร์โทร, ตำแหน่งที่สนใจ, วันที่ว่าง
- Footer พร้อมที่อยู่และ Google Maps embed

สไตล์: อบอุ่น เป็นกันเอง ไม่เป็นทางการ สีโทนส้ม-ขาว
เทคนิค: HTML/CSS/JS ไฟล์เดียว, responsive mobile-first, ฟอร์มเชื่อม Google Sheets
```

**ผลที่ได้ใน 1 ชั่วโมง:**

- Landing page ภาษาไทยทั้งหน้า
- Responsive — เปิดจากมือถือได้สวยงาม
- ฟอร์มสมัครงาน เชื่อมกับ Google Sheets อัตโนมัติ
- Load เร็ว ไม่มี bloat

## ขั้นตอนที่ทำจริงๆ

```
1. เขียน brief เป็นภาษาคน (ราว 10-15 ประโยค)
   ↓
2. ส่งให้ Claude Code — ได้ HTML + CSS + JS ครั้งเดียว
   ↓
3. เปิดไฟล์ในเบราว์เซอร์ ดูว่าตรงที่อยากได้ไหม
   ↓
4. แก้ไขจุดที่ไม่ถูกใจ — บอก AI ต่อเนื่อง
   ↓
5. Deploy ขึ้น server
   ↓
6. ส่งลิงก์ให้ลูกค้า
```

ทั้งหมดนี้ใช้เวลาไม่ถึง 60 นาที

## Deploy ขึ้นเว็บใน 2 นาที

**ง่ายที่สุด — Netlify drag-drop (ฟรี):**
1. ไปที่ [netlify.com](https://netlify.com) แล้ว sign up
2. ลาก folder หรือไฟล์ HTML ไปวางในหน้า dashboard
3. ได้ลิงก์ `.netlify.app` ทันที

**สำหรับคนใช้ terminal:**
```bash
# ติดตั้ง Vercel CLI ครั้งแรก
npm install -g vercel

# Deploy จาก folder ที่มีไฟล์ HTML
npx vercel --prod
```

ได้ URL ที่ใช้งานได้จริงใน 30 วินาที

<Callout variant="info" title="ไม่จำเป็นต้องมี server ตัวเอง">
Netlify และ GitHub Pages รับ deploy ฟรี ลาก HTML ไฟล์ขึ้นไป ได้ลิงก์ทันที ไม่มีค่าใช้จ่ายรายเดือน เหมาะมากสำหรับ landing page ที่ไม่มี backend
</Callout>

## สิ่งที่ AI ทำได้ดีกว่าที่คิด

### Content ภาษาไทย
AI เขียน copywriting ภาษาไทยได้ดีมาก — ตรง tone ที่อยากได้ ไม่ต้องเขียนเองทุกคำ แค่บอกใน prompt ว่า "เขียนให้เป็นกันเอง อบอุ่น เหมือนคุยกับน้อง"

### Mobile-first อัตโนมัติ
ถ้าใส่ว่า "responsive mobile-first" ใน prompt AI จะ generate CSS ที่รองรับมือถือมาให้เลย

### แก้ไขได้ทีละจุดง่ายมาก

บอก AI ตรงๆ แบบนี้ได้เลย:

```
เปลี่ยนสีปุ่ม "สมัครงาน" เป็น #FF6B35
```

```
เพิ่ม section รีวิวจากพนักงานปัจจุบัน 3 คน
ใช้ชื่อสมมติ เขียนให้ดูน่าเชื่อถือ เป็นกันเอง
```

```
ย้าย section โปรโมชั่นขึ้นไปอยู่เหนือ section ติดต่อเรา
และเพิ่มปุ่ม LINE สีเขียวด้านล่างฟอร์ม
```

แต่ละคำสั่ง AI แก้ให้ได้ทันที ไม่ต้องรู้ HTML เลย

## ลูกค้าได้อะไรจากเว็บไซต์ 1 ชั่วโมง

ลูกค้ารายนั้นได้รับใบสมัครงานภายใน 3 วันหลังส่งลิงก์ออกไป

**เทียบกับแบบเดิม:**
- เดิม: รอ 2-3 สัปดาห์ + ค่าใช้จ่ายสูง
- ใหม่: พร้อมใน 1 ชั่วโมง + ค่าใช้จ่ายแทบศูนย์

<Callout variant="success" title="Key Takeaway">
เว็บไซต์ที่ดีไม่จำเป็นต้องใช้เวลานาน สิ่งสำคัญคืออธิบายให้ชัดว่าต้องการอะไร ให้ใคร และเพื่ออะไร AI จะเติมส่วนที่เหลือให้เอง
</Callout>

## กรณีที่เหมาะกับ AI Website

- **Landing page** สำหรับโปรโมชั่น/อีเวนต์ชั่วคราว
- **หน้าสมัครงาน** หรือรับสมาชิก
- **Portfolio** แนะนำตัว/แนะนำธุรกิจ
- **หน้า FAQ** ตอบคำถามที่ลูกค้าถามบ่อย
- **Microsite** สำหรับสินค้าตัวใหม่

## กรณีที่ยังต้องการคนจริงๆ

ถ้าต้องการ **e-commerce** ที่มีตะกร้าสินค้า การชำระเงิน หรือ backend ซับซ้อน — AI ช่วยได้ส่วนหนึ่งแต่ยังต้องการ developer มาจัดการระบบหลังบ้าน

แต่สำหรับ **landing page**, **brochure site**, และเว็บเพื่อ **generate leads** — 1 ชั่วโมงก็พอ
