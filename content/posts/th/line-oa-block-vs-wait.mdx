## Open Rate 94.6% — แล้วทำไมลูกค้ายัง block?

LINE มีผู้ใช้ **56 ล้านคน** ในไทย — คิดเป็น 90.6% ของ internet users ทั้งหมด คนไทยใช้ LINE เฉลี่ย **7 ชั่วโมง 15 นาทีต่อเดือน** และเมื่อมีข้อความเข้า LINE OA มี **open rate 94.6%** สูงกว่า Email Marketing เกือบ 5 เท่า

ตัวเลขเหล่านี้ดูดีมาก ใช่ไหม?

แต่ถ้าตัวเลขดีขนาดนี้ แล้วทำไม LINE OA ของร้านส่วนใหญ่ถึง **โดน block** ทันทีที่ส่ง broadcast?

คำตอบไม่ใช่เรื่อง "ความถี่" ไม่ใช่เรื่อง "เวลาที่ส่ง" และไม่ใช่เรื่อง "รูปสวยหรือเปล่า" มันเป็นเรื่องของ **สิ่งที่ลูกค้าได้รับ เทียบกับสิ่งที่ลูกค้าคาดหวัง**

## 2 ฝั่งที่ต่างกันลิบ

| | ร้านที่ลูกค้า Block | ร้านที่ลูกค้ารอ |
|---|---|---|
| **เนื้อหา** | โปรโมชั่นซ้ำๆ ทุกครั้ง | ผสมหลายแบบ — เมนูใหม่, เบื้องหลัง, เคล็ดลับ, ดีลพิเศษ |
| **ความรู้สึกลูกค้า** | "ร้านนี้แค่อยากขายของ" | "ร้านนี้รู้ว่าฉันสนใจอะไร" |
| **ความถี่** | ส่งทุกวัน ทุกอย่างเท่ากัน | ส่งเมื่อมีเรื่องน่าสนใจจริงๆ |
| **Exclusive** | ทุกคนได้ข้อความเดิม | ลูกค้า LINE รู้สึกว่า "ได้สิทธิ์พิเศษ" |
| **ประโยชน์ต่อผู้รับ** | ประโยชน์ต่อร้านเป็นหลัก | มีคุณค่าสำหรับลูกค้าจริงๆ |
| **อารมณ์ที่เกิด** | รำคาญ | ตื่นเต้น อยากเปิดอ่าน |

## 5 สิ่งที่ทำให้โดน Block

### 1. ส่งแต่ "Sale Sale Sale"

ข้อความส่วนใหญ่ที่ทำให้โดน block มีหน้าตาแบบนี้:

> "โปรโมชั่นพิเศษ! ซื้อ 1 แถม 1 วันนี้วันเดียวเท่านั้น!!!"

ครั้งแรกยังโอเค ครั้งที่สองเริ่มรำคาญ ครั้งที่สาม block

ปัญหาไม่ใช่ "มีโปรโมชั่น" แต่คือ **ทุกข้อความคือโปรโมชั่น** ลูกค้าเริ่มอ่านออกว่า "เปิด LINE OA = ถูกขายของ" และสมองก็ตัดสินใจ: block เถอะ

### 2. ส่งถี่เกินไปโดยไม่มีคุณค่า

ส่งทุกวันโดยที่ข้อความไม่มีอะไรใหม่ ลูกค้าไม่ได้รู้สึกว่าร้าน active ลูกค้ารู้สึกว่าร้าน **รบกวน**

ความถี่ที่ "ดี" ไม่ใช่ตัวเลขตายตัว มันขึ้นอยู่กับว่าสิ่งที่ส่งมี **เหตุผลพอที่จะส่ง** หรือเปล่า

### 3. ข้อความสำหรับทุกคน ไม่ใช่สำหรับฉัน

"สวัสดีลูกค้า Food Stock ทุกท่าน!"

ประโยคนี้ไม่ได้ผิดอะไร แต่มันทำให้ลูกค้ารู้สึกว่าตัวเองเป็น **แค่หนึ่งในหมื่น** ไม่ใช่คนพิเศษ ผลคือข้อความดูเหมือน spam แม้จะไม่ใช่

### 4. ไม่มีเรื่องที่ "ให้" เลย

ทุกข้อความเป็นการ "ขอ" — ขอให้มาซื้อ ขอให้ใช้คูปอง ขอให้แชร์

ร้านที่ลูกค้ารอ มักจะ **ให้ก่อนขอ** — ให้ข้อมูลที่มีประโยชน์ ให้ content ที่สนุก ให้ความรู้สึกว่าการ add LINE OA ร้านนี้คุ้มค่า

### 5. ไม่รู้จักลูกค้าเลย

ส่งเมนูอาหารทะเลไปให้คนแพ้อาหารทะเล ส่งโปรโมชั่นเที่ยงไปให้คนที่ไม่เคยซื้อกลางวันเลย สิ่งเหล่านี้ไม่ได้ทำให้ลูกค้า angry แต่ทำให้ลูกค้ารู้ว่า **ร้านไม่รู้จักฉันเลย** และนั่นก็พอที่จะ block

<Callout variant="warning" title="ตัวเลขที่น่ากังวล">
ร้านอาหารที่ส่ง mass broadcast แบบเดิมทุกครั้ง มี open rate จริงๆ ที่ต่ำกว่า 10% แม้ LINE จะรายงาน "delivered" ครบ — เพราะลูกค้าเห็นชื่อร้านก็ไม่เปิดอยู่แล้ว
</Callout>

## 5 สิ่งที่ทำให้ลูกค้ารอข้อความ

### 1. เมนูใหม่ก่อนใคร

"พรุ่งนี้เราจะเปิดตัว [เมนูใหม่] เป็นครั้งแรก — ลูกค้า LINE OA ได้ลองก่อนใคร!"

นี่คือ exclusive ที่ง่ายที่สุดและ work ที่สุด ลูกค้าได้รู้สึกว่าการเป็น "สมาชิก LINE OA" มีประโยชน์จริงๆ ไม่ใช่แค่เปิดไว้ให้โดนขาย

### 2. เบื้องหลังที่ทำให้รู้สึกใกล้ชิด

"วันนี้พ่อครัวทดลองสูตรใหม่อยู่ครับ ล้มเหลวไปแล้ว 3 รอบ แต่รอบที่ 4 นี่ใกล้แล้ว — ถ้าสำเร็จจะเอามาขายวันศุกร์"

ข้อความแบบนี้ไม่ได้ขายอะไรเลย แต่มันสร้าง **connection** ลูกค้ารู้สึกว่าเป็นส่วนหนึ่งของเรื่องราว และอยากติดตามว่าจะเกิดอะไรขึ้น

### 3. ดีลที่มีแค่ใน LINE OA

ไม่ได้หมายถึง "ส่วนลด 10%" แต่คือสิ่งที่หาไม่ได้จากที่อื่น:

- ของแถมที่ไม่ได้ประกาศบน Facebook
- เมนู secret ที่สั่งได้เฉพาะคนที่รู้
- Early bird สำหรับ event พิเศษ
- ราคา members-only

เมื่อลูกค้ารู้ว่า "ของดีอยู่ใน LINE OA" พวกเขาจะไม่กด block

### 4. ข้อมูลที่มีประโยชน์จริงๆ (ไม่ขาย)

ร้านอาหาร — ส่งเคล็ดลับเลือกวัตถุดิบ ส่งสูตรง่ายๆ ทำที่บ้าน ส่งเมนูแนะนำสำหรับวันฝนตก

ร้านคาเฟ่ — ส่งวิธีชงกาแฟบ้านให้อร่อย ส่งแนะนำ playlist ทำงานวันนี้

เนื้อหาแบบนี้ไม่ได้ขายอะไรเลย แต่มันบอกลูกค้าว่า **ร้านนี้คิดถึงฉัน ไม่ใช่แค่คิดถึงเงินในกระเป๋าฉัน**

### 5. Timing ที่ตรงกับชีวิต

ข้อความโปรโมชั่นกลางดึก = รบกวน
ข้อความ "เมนูพิเศษวันนี้" ตอน 10 โมงเช้า = ช่วยตัดสินใจมื้อเที่ยง

การส่งข้อความ **เวลาที่ถูก** ให้กับ **คนที่ใช่** ทำให้ข้อความเดิมกลายเป็นคนละเรื่องกัน

<Callout variant="insight" title="ตัวเลขที่น่าสนใจ">
ร้านอาหารที่มี LINE OA ที่ active และส่งเนื้อหาที่หลากหลาย (ไม่ใช่แค่โปรโมชั่น) มี repeat purchase rate **สูงกว่า 2-3 เท่า** เมื่อเทียบกับร้านที่ไม่มี LINE OA หรือมีแต่ไม่เคยส่งอะไร
</Callout>

## ความจริงที่ทำให้ LINE OA ทำงานได้ — คือ "คุณค่า" ไม่ใช่ "ความถี่"

LINE OA ไม่ใช่ช่องทางสำหรับ "ยิง broadcast" — มันคือ **ช่องทางความสัมพันธ์**

คิดแบบนี้: ถ้ามีเพื่อนสนิทส่งข้อความมาหาคุณทุกวัน คุณรู้สึกยังไง? รู้สึกดีใช่ไหม ถ้าข้อความนั้นมีเรื่องน่าสนใจ

แต่ถ้าเพื่อนคนเดิมส่งข้อความมาทุกวันเพื่อขายของ ทุกครั้ง ไม่มีเว้น — คุณคงบล็อกเขาไปนานแล้ว

LINE OA ก็ทำงานแบบเดียวกัน

ประเด็นไม่ใช่ว่าส่งกี่ครั้ง แต่คือ **ทุกครั้งที่ส่ง ลูกค้าได้อะไรกลับไปบ้าง**

## Checklist ตรวจ LINE OA ตัวเอง (ง่ายๆ 5 ข้อ)

ก่อนจะส่ง broadcast ครั้งต่อไป ลองถามตัวเองด้วย 5 ข้อนี้:

- [ ] **ถ้าฉันเป็นลูกค้า — อยากรับข้อความนี้ไหม?**
- [ ] **ข้อความนี้ให้อะไรลูกค้า หรือแค่ขอให้ลูกค้าซื้อ?**
- [ ] **มีเหตุผลที่ดีพอสำหรับการส่งตอนนี้ หรือส่งเพราะ "นานแล้วยังไม่ได้ส่ง"?**
- [ ] **ถ้าลูกค้า 3 คนต่างกันได้รับข้อความนี้ ทั้ง 3 คนจะรู้สึกว่ามันเกี่ยวกับเขาไหม?**
- [ ] **ถ้าลบโลโก้ร้านออก ยังเดาได้ไหมว่าร้านไหนส่ง?** (ถ้าไม่ได้ = ข้อความ generic เกิน)

5 ข้อนี้ไม่ได้การันตีว่าข้อความจะ perfect แต่มันจะช่วยกรอง broadcast ที่ทำให้โดน block ออกไปได้เยอะมาก

## สิ่งที่ต้องทำได้เองวันนี้เลย

ไม่ต้องรอระบบซับซ้อน — เริ่มจาก 3 อย่างนี้ได้เลย:

**1. ตั้ง Greeting Message ให้ดูดี**
ลูกค้า add LINE OA ใหม่จะได้รับ greeting message ทันที — ส่วนใหญ่ร้านเปิดค้างไว้เป็น default หรือ "ขอบคุณที่ add นะคะ" ลองเปลี่ยนเป็นข้อความที่บอกว่า **ลูกค้าจะได้อะไรพิเศษจากการเป็น LINE OA friend**

**2. ส่ง broadcast ครั้งต่อไปด้วย "เรื่องเล่า" ไม่ใช่ "โปรโมชั่น"**
แทนที่จะ: "ซื้อ 1 แถม 1 วันนี้เท่านั้น!" ลองเปลี่ยนเป็น: "วันนี้วัตถุดิบชุดพิเศษเข้ามา พ่อครัวอยากทำเมนู X ให้ลอง — มีแค่ 20 ที่นั่งนะครับ"

**3. ดูตัวเลข Block ของเดือนที่แล้ว**
LINE OA Manager มีรายงานว่าแต่ละ broadcast มีคน block กี่คน ลองเปิดดู ถ้าทุก broadcast มี block เพิ่ม — นั่นคือสัญญาณที่ต้องเปลี่ยนแนวทาง

<Callout variant="info" title="กำแพงที่จะเจอต่อจากนี้">
3 ขั้นข้างต้นทำได้วันนี้เลย — และมันจะช่วยได้มาก แต่ถ้าอยากได้ผลแบบที่ร้านที่ลูกค้า "รอข้อความ" ทำได้จริงๆ สิ่งที่ต้องการคือ segmentation — ส่งข้อความต่างกันให้ลูกค้าต่างกลุ่ม ตามพฤติกรรม ตามยอดซื้อ ตามความถี่ที่มา ซึ่งต้องใช้ LINE OA Pro + ความเข้าใจ CRM + ระบบวางแผนเนื้อหาที่ชัดเจน
</Callout>

## สรุป: LINE OA ที่ดีไม่ใช่เรื่องเทคนิค — มันเรื่อง mindset

ร้านที่ลูกค้า block คิดว่า LINE OA คือ "ช่องทาง broadcast โปรโมชั่น"

ร้านที่ลูกค้ารอข้อความคิดว่า LINE OA คือ "ช่องทางดูแลลูกค้าที่ใกล้ชิดที่สุด"

ความแตกต่างนี้ไม่ได้ต้องการเงินเพิ่ม ไม่ต้องการทีมใหม่ แค่ต้องการ **เปลี่ยนวิธีคิด** ว่า LINE OA ของร้านมีไว้เพื่ออะไร

ลูกค้าที่ add LINE OA ร้านคุณ ให้โอกาสคุณแล้ว — คำถามคือคุณจะทำอะไรกับโอกาสนั้น
