## OpenAI เริ่มซื้อสื่อแล้ว

OpenAI ประกาศซื้อ **TBPN (Technology Business Programming Network)** รายการ tech podcast ที่ The New York Times เรียกว่า "Silicon Valley's newest obsession"

## TBPN คืออะไร

- **Daily live tech talk show** ออกอากาศวันจันทร์-ศุกร์ 11am-2pm PT
- Host โดย **John Coogan** และ **Jordi Hays**
- แขกรับเชิญระดับ **Mark Zuckerberg**, **Satya Nadella**, **Sam Altman**
- ผู้ชม ~70,000/episode — ตัวเลขไม่สูง แต่ **influence สูงมาก** ในวงการ tech
- **รายได้ $5M** ปีที่แล้ว คาดว่าปีนี้จะทำ **$30M+**

## เงื่อนไขดีล

- TBPN จะช่วย OpenAI ด้าน **communications และ marketing**
- แต่จะรักษา **editorial independence** — เลือกแขกเอง ตัดสินเนื้อหาเอง
- Jordi Hays ยืนยัน: "เราเคยวิจารณ์ industry ตลอด และจะทำต่อ"

## เทรนด์ที่ใหญ่กว่า — Tech Giants เป็นเจ้าของสื่อ

| ใคร | ซื้ออะไร |
|-----|---------|
| Jeff Bezos | The Washington Post |
| Marc Benioff | Time Magazine |
| OpenAI | TBPN |

ทุกครั้งที่ tech company ซื้อสื่อ คำถามเดียวกันเสมอ: **จะ independent จริงหรือ?**

## มุมกลับ — ทำไม OpenAI ต้องการสื่อ

- **ควบคุม narrative** ในยุคที่ AI ถูกวิพากษ์วิจารณ์หนัก
- **เข้าถึง audience** ที่เป็น tech decision makers โดยตรง
- **Creator economy** กำลังแซง legacy media — podcast มีอิทธิพลมากกว่าหนังสือพิมพ์
- **Communications strategy** ไม่ต้องพึ่ง PR firm ภายนอก

## สำหรับ SME ไทย

- **Media = Power** — ใครคุม content ใครคุม narrative
- **Podcast ยังเป็นช่องทางที่ underrated** ในไทย — ถ้าคุณมี expertise ในอุตสาหกรรมหนึ่ง podcast อาจมีค่ามากกว่าโฆษณา
- **AI companies กำลังกลายเป็น media companies** — ไม่ใช่แค่ขาย API อีกต่อไป
