## งาน "เปิดเว็บดูข้อมูล" กินเวลาคุณมากแค่ไหน?

![Cowork EP4 — Chrome Integration เลขาส่วนตัวบนเว็บ](/images/blog/cowork-ep4.jpg)

ถ้าคุณเคยทำสิ่งเหล่านี้เอง:

- เปิดเว็บคู่แข่ง 10 ร้าน ดูราคาทีละตัว จด Excel เอง
- กรอก Google Form ซ้ำๆ 20-30 rows เพราะข้อมูลมาจาก spreadsheet
- Research ข้อมูลธุรกิจ กด scroll อ่าน copy ข้อมูลใส่ doc ทีละชิ้น

คุณกำลังทำงานที่ **ไม่ต้องใช้สมองคุณเลย**

ผมเคยใช้เวลา 2 ชั่วโมงนั่งเช็คราคาคู่แข่ง 50 รายการให้ลูกค้า ทำด้วยมือ เปิดเว็บทีละหน้า จดทีละตัว

พอลอง Chrome integration ใน Cowork ครั้งแรก งานนั้นใช้เวลา **10 นาที**

---

## Chrome Mode คืออะไร?

ใน Claude Cowork มี mode พิเศษที่ให้ AI "ควบคุม browser" ได้จริงๆ:
- เปิด URL
- อ่านเนื้อหาในหน้าเว็บ
- คลิกปุ่ม
- กรอกข้อมูลในฟอร์ม
- Scroll หาข้อมูล
- ดึงข้อมูลออกมาเป็นตาราง

ทั้งหมดนี้ AI ทำเอง คุณแค่บอกว่าอยากได้อะไร

<Callout variant="info" title="Chrome Mode vs Browser Tool">
Chrome Mode ใน Cowork ต่างจาก "Browse" ปกติตรงที่สามารถ interact กับหน้าเว็บได้จริง — กดปุ่ม, กรอกฟอร์ม, navigate ข้ามหน้า ไม่ใช่แค่ "อ่าน"
</Callout>

## วิธีเปิดใช้งาน

### ขั้นที่ 1 — ติดตั้ง Claude Extension สำหรับ Chrome

ไปที่ Chrome Web Store → ค้นหา "Claude" → ติดตั้ง extension ของ Anthropic

### ขั้นที่ 2 — เปิด Cowork

เปิด Claude Desktop app → เลือก Cowork mode → สร้าง Task ใหม่

### ขั้นที่ 3 — บอก Task ที่ต้องการข้อมูลจากเว็บ

พิมพ์ task โดยบอก URL หรือชื่อเว็บที่ต้องการ Claude จะใช้ Chrome เปิดให้อัตโนมัติ

### ขั้นที่ 4 — กด Allow เมื่อ Claude ขอสิทธิ์

Claude จะขอ permission ก่อนเข้าถึงเบราว์เซอร์ กด Allow และดูมันทำงาน

<Callout variant="warning" title="ต้องมี Chrome เปิดอยู่">
Extension ต้องการ Chrome เปิดอยู่ขณะ Claude ทำงาน ถ้าปิด Claude จะหยุดกลางคัน
</Callout>

## ตัวอย่างที่ 1 — เช็คราคาคู่แข่ง 50 รายการใน 10 นาที

นี่คือ use case ที่ผมทำจริงให้ลูกค้า vape shop

**สิ่งที่ต้องทำ:** เช็คราคาสินค้า 50 SKU จาก 5 เว็บคู่แข่ง

**ก่อนหน้านี้:** 2 ชั่วโมง เปิดเว็บ 5 แท็บ ค้นหาทีละสินค้า copy ราคาลง Excel เอง มีโอกาสพิมพ์ผิดตลอด

**Task ที่ผมให้ Cowork:**

> "เช็คราคาสินค้าต่อไปนี้จากเว็บ [URL1], [URL2], [URL3]:
> 1. [ชื่อสินค้า 1]
> 2. [ชื่อสินค้า 2]
> ...
> ใส่ผลลัพธ์เป็นตาราง: ชื่อสินค้า | เว็บ A | เว็บ B | เว็บ C | เว็บเราราคาต่ำ/สูงกว่ากี่ %"

**ผลลัพธ์:** ตาราง Excel ครบ 50 รายการ พร้อม % เปรียบเทียบ ใน 10 นาที ไม่ต้องแตะ keyboard สักครั้ง

สิ่งที่ไม่ต้องทำอีก: เปิด-ปิดแท็บซ้ำ, copy-paste ราคา, คำนวณ % เอง

---

## ตัวอย่างที่ 2 — กรอก Google Form 20 Entries ใน 5 นาที

ลูกค้าคนหนึ่งต้องส่ง order เข้าระบบ supplier ผ่าน Google Form ทุกสัปดาห์ 20-30 rows ทำมือ = 30-40 นาที

**Task:**

> "กรอก Google Form นี้: [URL]
> ข้อมูลอยู่ใน spreadsheet นี้: [path ไฟล์ใน computer]
> ใส่ข้อมูลจาก column A (ชื่อสินค้า), B (จำนวน), C (วันที่ต้องการ) ทีละ row จนครบ 20 rows"

Claude เปิด Form, อ่านข้อมูลจาก spreadsheet, กรอกทีละ row, กด Submit ทุกครั้ง

**เวลาที่ใช้:** 5 นาที แทน 35-40 นาทีเดิม

## ตัวอย่างที่ 3 — Research ข้อมูลแบรนด์คู่แข่ง

ก่อนเริ่มโปรเจ็คให้ลูกค้าใหม่ ผมต้องทำ competitive research — ดูว่าคู่แข่งทำอะไรบ้าง ราคาอยู่ที่ไหน ขาย channel ไหน

**Task:**

> "Research ข้อมูลคู่แข่ง 5 เจ้าของ [ประเภทธุรกิจ] ในไทย
> สำหรับแต่ละเจ้า ดึงข้อมูล:
> - Website URL
> - สินค้า/บริการหลัก
> - Range ราคา
> - Social media channels + จำนวน followers โดยประมาณ
> - Unique selling point (ถ้าระบุไว้ชัด)
>
> รวบรวมเป็น summary 1 หน้า"

Claude เปิดเว็บ เช็ค IG/FB ของแต่ละแบรนด์ อ่านหน้า About/Product แล้วสรุปออกมา

**ก่อนหน้า:** 1-2 ชั่วโมง (เปิดหลายแท็บ อ่านเอง จด เอง)
**หลัง:** 15-20 นาที ได้ draft ที่ครบ แค่ต้องอ่าน verify

## ข้อควรระวัง — อย่าทำสิ่งเหล่านี้

### 1. อย่า Login บัญชีสำคัญในระหว่าง Chrome Mode ทำงาน

Claude ควบคุม browser ได้จริง ถ้าคุณมีบัญชีธนาคาร บัญชี business สำคัญ หรือ email ส่วนตัวค้างอยู่ใน session เดียวกัน มีความเสี่ยงที่ AI อาจ navigate ไปผิดหน้าได้

**แนวทาง:** ใช้ Chrome profile แยกต่างหากสำหรับ Cowork โดยเฉพาะ ไม่มีบัญชีสำคัญ login ไว้

### 2. เว็บที่มี CAPTCHA หรือ Anti-bot อาจขัดได้

บาง website detect ว่ามี automation และบล็อก Claude จะหยุดและรายงานให้คุณทราบ แต่ไม่สามารถผ่านได้เอง

### 3. เว็บที่เปลี่ยน layout บ่อย อาจต้องบอก Task ใหม่

ถ้าเว็บอัพเดท UI ชื่อ button หรือ layout เปลี่ยน Claude อาจหาไม่เจอ ต้องอัพเดท task เอง

### 4. ข้อมูลที่ดึงมาต้อง verify เอง

Claude อ่านเว็บได้ดี แต่ไม่ได้ถูกต้อง 100% โดยเฉพาะถ้าข้อมูลในเว็บมีหลาย version หรือแสดงราคาแบบ dynamic ต้อง spot check ก่อนใช้งานจริง

## สรุป: เลขาที่ทำงานซ้ำไม่รู้จักเหนื่อย

Chrome Mode ใน Cowork คือ upgrade ที่เปลี่ยนวิธีทำงานกับข้อมูลบนเว็บ

งานที่เคยต้องใช้เวลา 2 ชั่วโมง เพราะ "เปิดเว็บดูเอง" — สรุปตอนนี้ใช้เวลา 10-20 นาที เพราะ Claude เป็นคนเปิดเว็บแทน

**3 งานที่เหมาะกับ Chrome Mode:**
1. เช็คราคาคู่แข่ง (ทำซ้ำรายสัปดาห์/รายเดือน)
2. กรอกฟอร์มหลายรายการจาก spreadsheet
3. Research ข้อมูลเบื้องต้น (สรุปแบรนด์, ดู profile คู่แข่ง)

**3 งานที่ยังต้องทำเอง:**
1. Login บัญชีสำคัญ
2. งานที่ต้องใช้ judgment (ตัดสินใจซื้อ, อนุมัติ order)
3. เว็บที่มี 2FA หรือ CAPTCHA
