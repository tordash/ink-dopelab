## งาน NVIDIA GTC คืออะไร

NVIDIA GTC (GPU Technology Conference) เป็นงานประชุม AI ที่ใหญ่ที่สุดในโลก

ไม่ใช่งาน startup ไม่ใช่งาน industry เล็กๆ

GTC คือที่ที่ CEO Jensen Huang ประกาศ GPU รุ่นใหม่ บริษัท AI ชั้นนำจากทั่วโลกมาแสดง partnership และ enterprise deal ขนาดใหญ่เกิดขึ้น

ปี 2026 มีบริษัทจากหลายประเทศในเอเชียขึ้นเวที แต่มีบริษัทไทยเดียว — True Corp

## True Corp ขึ้นเวทีในฐานะอะไร

นี่คือส่วนที่น่าสนใจที่สุด

True Corp ไม่ได้ขึ้นเวทีในฐานะบริษัทโทรคมนาคม ไม่ใช่ "True Move H ผู้ให้บริการ 5G รายใหญ่"

True Corp ขึ้นในฐานะ **AI Infrastructure company**

หมายความว่า True Corp กำลัง position ตัวเองว่าเป็นผู้สร้าง infrastructure สำหรับ AI ในไทย ไม่ใช่แค่ผู้ให้บริการ connectivity

## AI Infrastructure หมายถึงอะไร

ในบริบทของ NVIDIA GTC และงาน AI ระดับ enterprise "AI Infrastructure" หมายถึง:

**Data Center ที่รัน AI** — GPU clusters ขนาดใหญ่สำหรับ training และ inference AI models

**Connectivity สำหรับ AI workloads** — Network ที่รองรับ data transfer สำหรับ AI training ที่ต้องการ bandwidth สูง

**AI Cloud Platform** — บริการ cloud ที่ปรับแต่งมาสำหรับ AI โดยเฉพาะ รวมถึง ML platform, model hosting, และ API สำหรับนักพัฒนา

**Edge AI** — การประมวลผล AI ใกล้กับ end-user ผ่านโครงข่าย 5G ของ True

<Callout variant="info" title="บริบทของ True Corp">
True Corp ควบรวมกับ DTAC เมื่อปี 2023 กลายเป็น operator รายใหญ่อันดับ 2 ของไทย โครงข่าย 5G ที่ครอบคลุมในไทยทำให้มี infrastructure พื้นฐานสำหรับ edge AI deployment ทั่วประเทศ
</Callout>

## Partnership กับ NVIDIA

การขึ้นเวที GTC มักหมายความว่ามี partnership กับ NVIDIA ในระดับที่สำคัญ

รายละเอียดที่รายงานออกมา: True Corp กำลังลงทุนใน NVIDIA GPU infrastructure เพื่อสร้าง AI Data Center ในไทย โดยน่าจะเป็นส่วนหนึ่งของ True Digital Park ที่ Bangkok

นี่หมายความว่า developer และองค์กรไทยที่ต้องการ AI compute อาจมีทางเลือกในประเทศแทนที่จะต้อง access cloud จากสิงคโปร์หรือญี่ปุ่น

## ทำไมเรื่องนี้สำคัญสำหรับไทย

**Data Sovereignty** — ข้อมูล sensitive ของธุรกิจและรัฐบาลไทยจะสามารถ process ในประเทศได้ ไม่ต้องส่งออกไปต่างประเทศ

**Latency ที่ต่ำกว่า** — AI ที่รันใน data center ในไทยจะ response เร็วกว่า AI ที่ host ในสิงคโปร์สำหรับ user ในไทย

**เศรษฐกิจดิจิทัล** — AI infrastructure ในประเทศหมายถึง local developer มีทรัพยากรมากขึ้น startup ไทยมีโอกาสสร้าง AI product ที่แข่งขันได้

**ตำแหน่ง ASEAN Hub** — ถ้าไทยมี AI infrastructure ระดับ world-class อาจดึงดูด AI company จากทั่วโลกมา setup regional operations

## มุมมองจากผู้ประกอบการ

สำหรับ SME และ startup ไทย เรื่องนี้อาจยังดูไกลตัว แต่มีนัยระยะยาว

**ราคา AI service ในไทยจะถูกลง** — เมื่อมี local infrastructure แข่งขัน cloud provider ต่างชาติต้องลดราคา หรือ local option ให้ราคาที่แข่งขันได้

**Compliance ง่ายขึ้น** — กฎหมาย PDPA และกฎระเบียบเรื่อง data storage ที่กำลังจะเข้มขึ้น จะ comply ได้ง่ายกว่าถ้า data อยู่ในประเทศ

**Local AI product** — มีโอกาสให้ startup ไทยสร้าง AI product โดยใช้ local infrastructure ที่ถูกกว่า ซึ่งอาจเห็นผลใน 2-3 ปีข้างหน้า

## สิ่งที่ยังต้องติดตาม

การประกาศในงาน conference กับ execution จริงมักต่างกัน

สิ่งที่ต้องดูต่อ:
- Timeline ของการ build data center จริงๆ
- ราคาที่ True Corp จะคิดกับ developer และ enterprise
- API และ developer tools ที่จะเปิดให้
- Partnership กับ local AI startup

## สรุป

True Corp บนเวที NVIDIA GTC เป็น signal สำคัญว่า AI infrastructure ในไทยกำลังเป็นจริงมากขึ้น

ก่อนหน้านี้ "AI ในไทย" หมายถึงการใช้ tools และ service จากบริษัทต่างชาติ

ตอนนี้มีบริษัทไทยที่ลงทุนสร้าง infrastructure และยืนบน stage เดียวกับ NVIDIA

นั่นคือ signal ที่ไม่ควรมองข้าม
