## "Just another AI chatbot?"

If you've used ChatGPT or Claude.ai in the typical ask-and-answer format, you've probably noticed...

- You have to copy-paste data into it every time
- After getting a response, you copy results back into your files manually
- You can't actually tell it to "organize files" or "create an Excel sheet"
- Open a new chat and it forgets everything

**You're not using it wrong** — those tools were literally designed to just "answer questions."

But what if there was an AI that could **open files on your computer**, edit documents, create new files, browse the web — all just by telling it what you want in plain language?

That's **Claude Cowork**.

---

## What is Cowork? (3 Lines)

1. An **AI agent** that works **on your machine** — not just replies in a chat
2. You **point it to a folder** → it reads, edits, and creates files for you
3. Tell it what you need → **let it work** → come back and check the results

Think of it this way: ChatGPT is like **asking a knowledgeable friend**, but Cowork is like **hiring an assistant** who actually does the work.

## Get Started in 5 Steps

### Step 1 — Download the App

Go to [claude.com/download](https://claude.com/download) → choose Mac or Windows → install like any other app.

### Step 2 — Log In

Open the app → sign in with your Claude account (Pro plan or above).

### Step 3 — Switch to Cowork Mode

In the app you'll see **"Chat"** and **"Cowork"** options — select **Cowork**.

### Step 4 — Create Your First Task

Go to the **Tasks** tab → type what you want done, for example:

> "Organize the files in my Downloads folder by file type"

### Step 5 — Grant Folder Access

Claude will ask which folder to access → select the folder → **click Allow**

That's it. Claude starts working immediately.

## What Else Can Cowork Do?

Beyond file organization, here's what Cowork handles in practice:

### Create Documents
- Summarize meeting notes from saved files → create a PowerPoint
- Read CSV data → build an Excel report with charts
- Combine data from multiple files → produce a single summary document

### Search and Analyze
- Read a long PDF and summarize it
- Compare data across 2–3 files
- Find specific numbers in hundreds of pages of documents

### Browse the Web (Claude in Chrome)
- Research competitors and compile results into a table
- Extract data from websites and save as files
- Fill out forms, read emails, check information

### Automate on Schedule (/schedule)
- Every morning: summarize important emails → save to file
- Every week: compile sales data → generate a report
- Every Monday: check calendar → summarize the week's meetings

## Things to Know Before Using It

| Limitation | Details |
|---|---|
| **App must stay open** | Close the app = Claude stops working |
| **No cross-device sync** | Work history stays on that computer only |
| **Fresh start each session** | Claude doesn't remember previous sessions (but can read existing files) |
| **Delete requires approval** | Claude asks before deleting anything |
| **Not free** | Requires Pro ($20/month) or higher |
